ОПИСАНИЕ

1. Создать папку на рабочем столе "Origin_Repo"
2. Зайти на GH к преподователю https://github.com/Bulvinkoln/4162_group и For-кнуть его Repo к себе
3. Зайти в свой профиль GH и скопировать ссылку этого Repo
4. Далее командой "Git clone" склонировать Repo в папку
5. Далее командой "cd" перейти в клонированную папку, а именно "cd 4162_group/"
6. Проверить путь и убедиться что мы находимя именно в той папке  ~/Desktop/Origin_Repo/4162_group (main)
7. Создаем новую ветку с авто переходом к примеру git checkout -b "br2"
8. Создаем файл (уже на новой ветке) "README.md" или "config.txt" или похожий далее пишем описание проекта и схраняем CTRL+S
9. Добавляем файл в отслеживаемые командой add
8. Добавляем commit / save / сохранение командой "Git commit -am "add config"

ЗАПРОС НА СЛИЯНИЕ 

1. Командой Git push -u br2 отправляем запрос (название ветки обязательно, после того как GH запомнит нашу ветку, мы сможем отправлять изменения просто командой git push)
